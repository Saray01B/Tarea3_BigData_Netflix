# Tarea3_BigData_Netflix

El proyecto procesa un conjunto de datos sobre las peliculas y series de Netflix, con el objetivo de aplicar técnicas de limpieza, transformación y análisis exploratorio de datos (EDA) para obtener estadísticas como los géneros más frecuentes, distribución por año de lanzamiento, cantidad de títulos por tipo, entre otros. Este análisis es útil para optimizar recomendación y diseñar estrategias para mejorar la experiencia del usuario.

Detalles de las variables

ID: Identificador único del título
Tipo: Película o serie
Título: Nombre del contenido
Director: Director del título
Elenco: Actores y actrices principales
Fecha de agregación: Fecha en la que el contenido fue añadido.
Año de lanzamiento: Año de estreno del contenido
Clasificación: Clasificación por edades
Duración: Duración de la pelicula o número de temporadas de la serie
Género: Categorías de género asociadas al contenido
Descripción: Sinopsis del títulol
